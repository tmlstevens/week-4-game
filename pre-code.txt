EACH GAME
 - generate random goal number: 19-120
 - generate random number for each crystal: 1-12
 - player score = total of crystal values clicked
 - win if player score = goal number
 - lose if player score > random goal number
 - auto restart game after win or lose
 - wins and losses keep logging
